## Topic: Cases

### Sub-topic: Checklist

- Empty input
- Single item
- Duplicates
- Negative values
- Already sorted
- Maximum constraints

## Topic: Testing

### Sub-topic: Approach

Test normal cases first, then boundaries, then cases that stress the chosen invariant.

## Topic: Debugging

### Sub-topic: Method

Trace one failing example manually and compare actual state against expected invariant.
